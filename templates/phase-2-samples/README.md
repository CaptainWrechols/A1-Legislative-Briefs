# Phase 2 visual samples

**Authoritative sample:** `NV1-Issue-Brief-4-Growth-and-Water-Scarcity-v1.5.pdf` (in this folder).

`design-packager` mirrors this sample's **visual system** — colors, fonts, spacing, and module types — for the citizen legislative reality brief:

- White page; navy `#1A2D4F` titles, big stat numbers, table header rows; secondary navy `#2E4A78`
- Terracotta `#C0392B` ALL-CAPS letter-spaced section headers; soft pink tint `#E8D5D3` on accent bars
- Arial body (~9pt, `#444444`/`#666666`); Georgia serif on quote-style cards only
- Modules: eyebrow header, quote cards (thin gray border + terracotta top bar), stat cards (navy top bar + huge navy number), navy-header comparison tables with alternating rows, numbered question grid

Extracted tokens live in `config/forum-brand.yaml`.

**Rules:**

- Do **not** copy this sample's headings, titles, kicker text, or body text into the legislative reality brief. Look-and-feel only.
- The content differs on purpose: the sample presents conversation options; our brief maps legislative history (what was tried, what moved, what stalled, who showed up).
- If a newer official sample is added here, update `config/forum-brand.yaml` to match before packaging.
